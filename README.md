## Close the YouTube 'Continue Watching' mini-player

Clone this repository with `git clone git@github.com:Hypnotriod/youtube-close-mini-player.git`. Or just download the archive and extract it at `Code` -> `Download ZIP` .  
Go to extensions in your browser (e.g., `chrome://extensions/`).  
Enable the `Developer mode` if needed.  
Drag the whole folder onto the extensions page.
